//             ゲーム内容               
//--------------------------------------

//--------------------------------------
//          アピールポイント            
//--------------------------------------
  この作品は、「進め！キノピオ隊長」を基ととしたゲームを作りたいと思い制作を開始しました。
「進め！キノピオ隊長」とは、箱庭アドベンチャーゲームで、様々なギミックを駆使しながらゴールのスターを目指すゲームとなっており、
本作でもその仕様に近しい形で実装しました。
  坂道を上る処理や、地面(フィールド)に縮退ポリゴン(メッシュポリゴン)を使用したり、外部ファイルを数多く使用してマップの配置物やステータスを外部で管理できるようにすることを
今回の制作で尽力したポイントです。

//--------------------------------------
//                反省点                
//--------------------------------------
  本作の制作の反省点は、
・実装したいこととスケジュールが釣り合わなかったこと。
・当たり判定において、内積・外積を使った処理が十分にできなかったこと。
・本来の企画どおりに仕様を組み込めなかったこと。
・コントローラーの実装に大幅に遅れてしまったことと、スティックによる移動ができなかったこと         です。

  本来であれば、オブジェクトを使って穴を埋めて道を作ったり、別のオブジェクトを使って敵を攻撃したりしたかったのですが、実装時間とスケジュールがかみ合わず、
目標と違った形でゲームを作ってしまいました。
  また、当たり判定においても、当初は外積を使って壁の当たり判定を行い、次に坂道を上るのに外積、内積を使って登らせる、という目標を立てたのですが、
初の３Dということで、座標設定の概念や角度、初めての計算など色々つまづいてしまい、外積処理を放棄したり、坂道を登らせる方法は、
“ふもとから頂点までの距離の中で、プレイヤーはふもとからどれだけ頂点に近いのか”で割合を取り、
頂点の高さ × 割合でプレイヤーのY座標を設定していました。

次回以降の制作では、スケジュール通りに進められるよう、自分の実力を把握し、無理な高望みはせず、叶えられる範囲で自分の目標を叶え終えたうえで、次のステップに進もうと思いました。
